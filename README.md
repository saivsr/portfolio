Projects

A collection of internal tooling I designed, built, and ran in production, shared
here as work samples.
Each top-level folder in this repository is a self-contained project with its own
README covering what it does, how it's architected, and the engineering decisions
behind it. Open any project folder to dive in.
A note on sanitization
Across every project here, live client data, credentials, and identifying details
have been replaced with fictional placeholders. The architecture, logic, and
engineering decisions are real and unchanged — only the sensitive values are
stubbed; where something has been redacted, the project's README or code says so.
